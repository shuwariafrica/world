## sbt-world

The sbt plugin. It declares which locales and zones a build needs, slices the curated
dataset accordingly, and generates message catalogues from the project's own sources.

```scala
addSbtPlugin("africa.shuwari" % "sbt-world" % "@VERSION@")
```
