## world-data

The curated dataset compiled at build time. It is never placed on a runtime classpath: `sbt-world` reaches it through a hidden configuration and emits only the slices a build declares.

```scala
libraryDependencies += "africa.shuwari" %%% "world-data" % "@VERSION@"
```

This page is a placeholder. The documented surface arrives with the API it describes.
