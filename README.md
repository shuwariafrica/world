# Money Library (WIP)

A flexible, extensible Scala library for working with monetary values, currencies, and transactions.

## Features (Planned)

- Representation of money values and amounts
- Currency abstractions and ISO currency codes
- Transaction modeling and simple ledger operations
- Currency conversion and exchange rate support
- Locale-dependent formatting
- Error handling and improved type safety

## Modules

#### money

Core primitives, money values, currencies, conversion logic, and helpers.

#### locale

Locale and country-specific functionality
